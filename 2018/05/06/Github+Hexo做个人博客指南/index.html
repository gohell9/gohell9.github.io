Github+Hexo做个人网页指南

发表于 2018-05-06

一直想写个Hexo + Github 做个人博客的教程,但是人啊,一定学学会找轮子 .

Makedown作者約翰·格魯伯（John Gruber）一样的场面人除外

首先,附上官网指南

https://hexo.io/zh-cn/index.html
你要知道,官网上的文档永远是最新的.

前置条件:

1.github新建一个仓库,

比如你github主页是:https://github.com/xiaoming

所以,建个仓库: xiaoming.github.io

2.本地安装git, 并可以push到github某个仓库内容,哪怕是一个readme.md

3.安装 nodejs.msi

4.上面两个工具安装完整之后，打开 Git Bash ，只需要使用npm即可完成Hexo的安装。
$ npm install -g hexo-cli

开搞:

1.新建一个文件夹作为站点,比如:like

在like文件夹 右键 打开git bash:

hexo init blog
cd blog
git clone https://github.com/iissnan/hexo-theme-next themes/next # 下载一个next主题
npm install hexo-deployer-git –save # 下载安装部署工具

编辑blog内的_config.yml的几个值,比如

title: xiaoming的空间 //你博客的名字
author: xiaoming //你的名字
language: zh-Hans //语言 中文
theme: next //刚刚安装的主题名称
deploy:
type: git //使用Git 发布
repo: https://github.com/xiaoming/xiaoming.github.io.git // 刚创建的Github仓库

hexo clean

hexo g # 生成静态页面

hexo d # 部署到github上去

hexo s # 可以在本地:localhost:4000 预览博客效果
